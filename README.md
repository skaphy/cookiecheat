CookieClicker用のチートブックマークレットです。

このリンクをブックマークに入れた後、CookieClickerを開きブックマークに入れたリンクを開きます。

そうすると左上に白い半透明のものがでてくるのでそこにマウスを持って行くと使用出来るチートが表示されます。
